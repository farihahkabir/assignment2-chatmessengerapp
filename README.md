15 minute Chat App

Description: Chat application where people can join/start a chat session. Once the session starts, you can only talk for 15 minutes. You can also stop the session within 15 minutes.

Technologies used:
1. node.js
2. socket.io
3. mongoDB
4. Jquery
5. HTML
6. CSS
